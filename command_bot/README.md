# command_bot

`hello` の発展形。`@bot <コマンド名> <引数>` 形式の入力をパースして、コマンドごとに別の処理に振り分けるサンプル。

## できること

- `@bot help` / `@bot ヘルプ` … コマンド一覧
- `@bot search <質問>` … 検索コマンド（実装は TODO。RAG/LLM を差し込む場所が用意されている）
- `@bot summarize <文章>` … 要約コマンド（同上）
- `@bot echo <文字>` … 引数をそのまま返す
- `@bot ping` … 生存確認（`pong` を返す）
- メンション + 知らないコマンド → 警告メッセージ
- メンションだけ → 使い方を案内

## ポイント

- `strip_mention()` で `<at>bot</at>` メンション部分を除去
- `parse_command()` で先頭単語を **コマンド名**、残りを **引数** に分割
- `COMMANDS` ディクショナリで「コマンド名 → ハンドラ関数」のマップを管理。新コマンド追加は関数を書いてマップに登録するだけ

## 主なファイル

- `src/agent.py` … コマンドディスパッチのメイン
- `src/app.py` … aiohttp 起動
- `src/config.py` … 環境変数読み込み
