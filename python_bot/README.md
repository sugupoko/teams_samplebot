# python_bot

Microsoft Teams 上で動くシンプルな Python ボット。

## できること

- **通常メッセージ**: AI 応答を返す（現在はダミークライアント。Azure OpenAI に差し替え可能）
- **`@bot help` / `@bot ヘルプ`**: 使い方を表示
- **`@bot summary` / `@bot サマリー`**: これまでの会話履歴を表示
- **`@bot`（メンションのみ）**: 現在のチャット情報（会話 ID / 送信者 / チャンネル / 参加者一覧 / メッセージ）を返す
- 会話履歴はインメモリ保持（プロセス再起動でリセット）

## 主なファイル

- `src/agent.py` … メッセージハンドラとロジック
- `src/app.py` … aiohttp で起動
- `src/config.py` … 環境変数の読み込み
